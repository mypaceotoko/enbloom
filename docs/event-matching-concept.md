# ご縁会 / ConnectBloom Circle 構想

## ピボット方針

既存のイベント構想は、恋愛イベントではなく、**紹介者がひらく少人数の活動交流会** へピボットします。

ConnectBloom におけるご縁会は、活動・興味・共創・紹介を軸に、安心して話せる小さな交流の場です。

## 名称候補

- ご縁会
- ConnectBloom Circle
- コラボ会
- 小さな交流会
- 紹介者イベント

現時点の推奨表現は **ご縁会 / ConnectBloom Circle** です。

## コンセプト

**紹介者がひらく、少人数の活動交流会** です。

特徴:

- 招待コードや紹介ルートと紐づく。
- 共通の活動ジャンルや興味テーマを持つ人が集まる。
- 参加前にプロフィール、活動ジャンル、つながり方のスタンスを確認できる。
- イベント後に「話してみたい」を送り、相互ならコネクトする。
- コネクト後に会話が開放される。

## 交流会テーマ例

- AI開発好きの会
- ラジオ配信者交流会
- 怪談好きの会
- ダンス仲間の会
- ブログ作業会
- 映画を語る会
- ゲーム制作会
- 地域交流会

## 募集ボードとの接続

将来的には、ご縁会 / ConnectBloom Circle と募集ボードを接続します。

- 「ブログ作業会をしたい」募集から小さな交流会へつなげる。
- 「ラジオにゲストで出てくれる人募集」からコラボ会へつなげる。
- 「ゲーム制作仲間募集」から制作会へつなげる。

## 今回まだ実装しないこと

- イベント作成機能
- 投稿機能
- チケット販売
- 決済
- リアルタイム参加管理
- 募集ボード本実装
- ご縁会の本番運用

## 招待コードとの関係

**招待コード = ただの入場券ではなく、誰のご縁から来たかを記録する参加ルート** です。

ご縁会は、招待コードや紹介者に紐づいた小規模な活動交流会として扱います。現時点では実装しませんが、招待コード・紹介者・`introductions` は、この将来構想に接続できる前提で設計を維持します。

## 法務メモ

ご縁会は活動・興味・共創・紹介の交流会として設計します。ただし、イベント運営、本人確認、年齢確認、利用規約、各種法令への該当性は、公開前に専門家確認が必要です。
